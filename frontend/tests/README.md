Testing
=======

This project scaffold does not include a test runner by default. Common choices:

- Vitest (recommended for Vite + Vue 3)
- Jest (if preferred)

To add Vitest quickly:

1. npm install -D vitest @vue/test-utils
2. Create `vitest.config.ts` and add test scripts in package.json

Keep tests under `tests/` or `src/__tests__/`.
