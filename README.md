# Almoxarifado

Monorepo do sistema de almoxarifado com backend em Spring Boot e frontend em Vue 3 + Vite.

## Estrutura

- `backend/`: API REST, regras de negocio e persistencia.
- `frontend/`: aplicacao web e rotas da interface.

## Execucao local

### Backend

```cmd
cd backend
mvnw.cmd spring-boot:run
```

### Frontend

```cmd
cd frontend
npm install
npm run dev
```
